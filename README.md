# Visualize Industrial Trends



#### 데이터를 기반으로 한 산업 분석은 세계 경제를 선도하는 여러 선진국과 빅테크 기업들에서 진행하고 있다.
#### 수많은 데이터 더미에서 의미 있는 데이터를 찾아내는 과정을 경험해보고 싶어 데이터 분석을 통해 산업 트렌드를 조사해보는 프로젝트
